Some interesting tools, software, links, etc. I've found and I want to share.

ðŸ¤–Robostack

Robostack is a bundling of the Robot Operating System (ROS) for Linux, Mac and Windows using the Conda package manager (well, please use mambaforge!). They provide (early 2022) ROS packages for ROS1 Noetic, ROS2 Foxy and ROS2 Galactic (experimental), as well as a variety of ROS related plugins for Jupyter notebooks.

I'm using it on my new M1 mac and it works very well! Having a native ROS installation, easy to setup and maintain, is a great improvement! I'm now thinking to use it with our students for the ROS and experimental robotics teaching unit I work on.
